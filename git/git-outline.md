# Git 新手大纲

### 1. 核心概念

- **暂存区**
- **未追踪**
- **本地仓库**
- **远程仓库**
- **git日志**
- **分支**

### 2. 常用的git命令

* **add**
  * 将文件加入暂存区
* **commit**
  * 将文件提交到本地仓库
* **push**
  * 将本地仓库的文件修改推送到远程仓库
* **pull**
  * 拉取远程仓库的更新
* **log**
  * 查看git日志
* **reset**
  * 版本回退
* **cherry-pick**
  * 检出其他分支的某个提交到当前分支
* **rebase**
  * 保持一条流基线
  * 合并多个提交为一个
* **stash**
* tag
  * 给某次具有里程碑或特殊意义的提交打上tag标签
* rm
  * 文件删除
* mv
  * 文件移动
* diff
  * 文件更改比对
* init
  * git 仓库初始化（可针对空文件夹或非空文件夹）

### 3. 常见的业务场景

		* 文件冲突解决
		* 多分支切换
		* 保存修改到暂存区
		* 协作开发
		* 版本回退或切换
